# CVER Inc.

> **Delightful digital tools, made since 2011.** A small studio building native-Mac and web software for people who want to *own* their stuff, not rent it — the library you already own, the photos you already took, the browser you already use. Yours, offline, with source you can read.

Charming Vision, Endless Recreation. We build what we'd want to use ourselves, ship it open, and keep it sharp. We live with cats 🐱

## Native Mac

Runs on your machine, not someone's cloud.

- **[snapsift](https://github.com/CVERInc/snapsift)** — sift the near-duplicate snaps Apple's Photos detector misses, with a quality-aware keeper and a visual review pass.
- **[reepub](https://github.com/CVERInc/reepub)** — turn the scanned PDFs you own into clean, reflowable EPUB3, with on-device Apple Vision OCR.
- **[shelfseer](https://github.com/CVERInc/shelfseer)** — talk to the library you already own — notes, documents, books you bound yourself — 100% offline.
- **[sheersweep](https://github.com/CVERInc/sheersweep)** — the Mac cleaner you can read: open source, dry-run first, never touches your real files.
- **[signet](https://github.com/CVERInc/signet)** — the shared SwiftUI design system behind these apps.

## Developer tools

- **[clikae](https://github.com/CVERInc/clikae)** — one on-ramp for every AI coding CLI (Claude Code, Codex, Gemini, Antigravity): land back on your recent sessions across all your accounts.
- **[clioil](https://github.com/CVERInc/clioil)** — make shipping your code to any registry simple — native Mac, multi-ecosystem, friendly guidance.

## Web & craft

- **[bleedblend](https://github.com/CVERInc/bleedblend)** — zero-config Safari chrome tinting: paint the browser edges to match your page, across iPhone, iPad, and Mac.
- **[motifmint](https://github.com/CVERInc/motifmint)** — turn any image into a clean, recolorable SVG logo or icon, entirely in your browser. No upload, no server.
- **[seikyusho](https://github.com/CVERInc/seikyusho)** — invoice automation for Japanese businesses, built on Google Apps Script.

## Obsidian

- **[tugtile](https://github.com/CVERInc/tugtile)** — markdown-backed kanban for Obsidian. Tug tiles to reorder.
- **[marktile](https://github.com/CVERInc/marktile)** — the markdown editor pane from the same family. CJK-friendly.
